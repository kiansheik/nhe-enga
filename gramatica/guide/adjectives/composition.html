# Step-by-step Process

There are some rules to compositions which we need to take into account.

Let's define some terms for this process:

# Terminology

Sounds Types Letters
Oral Consonants p b t s x k r gû û î ŷ
Nasal Consonants m n ng nh mb nd
Oral Vowels a á e é i í y ý o ó u ú
Nasal Vowels ã ẽ ĩ ỹ õ ũ

Base root: Noun Modifier root: Adjective

# Steps

  1. If the base root ends in a

    tonic vowel á é í ý ó ú

    • replace it with its
      non-tonic counterpart a e i y o u

  2. If there is an encounter of

    Oral Consonants p b t s x k r gû û î ŷ

    • remove the base root's ending consonant

  3. If the base root ends in

    Any Consonant p b t s x k r gû û î ŷ m n ng nh mb nd
    and the modifier starts with a '(glottal stop)

    • remove the '(glottal stop) from the modifier

  4. If the base root ends with

    Any Nasal m n ng nh mb nd ã ẽ ĩ ỹ õ ũ<
    and the modifier starts with
    Any Consonant p b t s x k r gû û î ŷ m n ng nh mb nd

    1. Nasalize the final vowel of the base root, if not already marked

    2. If the base root ends in nh

      • it becomes a î

    3. If the base root ends in

      any other nasal consonant m n ng mb nd

      • delete it

    4. If the modifier root does not already contain

      any nasals m n ng nh mb nd ã ẽ ĩ ỹ õ ũ<

      • nasalize the first letter based on the following table, if needed:
      Oral Nasal
      p mb
      k ng
      t nd
      s nd

  5. After having considered the above cases, there is nothing left to do but put the base root and modifier root together {base}{modifier}

That's composition!
