 She is a beautiful woman using the "adjective" in composition with the base noun to create a new root

There are some rules to compositions which we need to take into account.

Let's define some terms for this process:

Oral Consonants p b t s x k r gû û î ŷ
Nasal Consonants m n ng nh mb nd
Oral Vowels a á e é i í y ý o ó u ú
Nasal Vowels ã ẽ ĩ ỹ õ ũ

  • If the root ends in a vowel with a

    tonic accent á é í ý ó ú
    then replace it with it's
    non-tonic counterpart a e i y o u

  • If there is an encounter of

    Oral Consonants p b t s x k r gû û î ŷ

    • Remove the base root's ending consonant

  • If the base root ends in

    Any Consonant p b t s x k r gû û î ŷ m n ng nh mb nd
    and the modifier starts with a ' (glottal stop)

    • Remove the ' (glottal stop) from the modifier

  • If the base root ends with

    Any Nasal m n ng nh mb nd ã ẽ ĩ ỹ õ ũ<
    and the modifier starts with
    Any Consonant p b t s x k r gû û î ŷ m n ng nh mb nd

    • Nasalize the final vowel of the base root, if not already marked
    • If the base root ends in nh then it becomes a î, otherwise the final
      Nasal Consonants m n ng mb nd
      will be completely removed
    • If the modifier root does not already contain
      any nasals m n ng nh mb nd ã ẽ ĩ ỹ õ ũ<
      • Nasalize the first letter based on the following table, if needed: | Oral | Nasal | |------|-------| | p | mb | | k | ng | | t | nd | | s | nd |

  • After having considered the above cases, there is nothing left to do but put the base root and modifier root together `{base}{modifier}

That's composition!
