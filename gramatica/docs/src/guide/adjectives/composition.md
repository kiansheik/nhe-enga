# Composition

Sometimes instead of saying something like _That woman is **beautiful**_ we will say _She is a **beautiful woman**_.

| Tupi | English | Explanation |
|------|-----------------|--------|
| <root root="kunhã" /> <root root=i entryNumber=4 /> <root root='poranga' /> | (that) woman is beautiful | using the _"adjective"_ as a second class verb |
| <root root=i entryNumber=4 /> <root root="kunhã" /><root type=noun root='poranga' /> | She is a beautiful woman | using the _"adjective"_ in composition with the base noun to create a *new root* |

There are some rules to compositions which we need to take into account.

Let's define some terms for this process:

<Tooltip content="Oral Consonants">p b t s x k r gû û î ŷ</Tooltip>
<Tooltip content="Nasal Consonants">m n ng nh mb nd</Tooltip>
<Tooltip content="Oral Vowels">a á e é i í y ý o ó u ú</Tooltip>
<Tooltip content="Nasal Vowels">ã ẽ ĩ ỹ õ ũ</Tooltip>

- If the root ends in a vowel with a <Tooltip content="tonic accent">á é í ý ó ú</Tooltip> then replace it with it's <Tooltip content="non-tonic counterpart">a e i y o u</Tooltip>
- If there is an encounter of <Tooltip content="Oral Consonants">p b t s x k r gû û î ŷ</Tooltip>
  - Remove the base root's ending consonant
- If the base root ends in <Tooltip content="Any Consonant">p b t s x k r gû û î ŷ m n ng nh mb nd</Tooltip> and the modifier starts with a `' (glottal stop)`
  - Remove the `' (glottal stop)` from the modifier
- If the base root ends with <Tooltip content="Any Nasal">m n ng nh mb nd ã ẽ ĩ ỹ õ ũ<</Tooltip> and the modifier starts with <Tooltip content="Any Consonant">p b t s x k r gû û î ŷ m n ng nh mb nd</Tooltip>
  - Nasalize the final vowel of the base root, if not already marked
  - If the base root ends in `nh` then it becomes a `î`, otherwise the final <Tooltip content="Nasal Consonants">m n ng mb nd</Tooltip> will be completely removed
  - If the modifier root does not already contain <Tooltip content="any nasals">m n ng nh mb nd ã ẽ ĩ ỹ õ ũ<</Tooltip>
    - Nasalize the first letter based on the following table, if needed:
    | Oral | Nasal |
    |------|-------|
    | p    | mb    |
    | k    | ng    |
    | t    | nd    |
    | s    | nd    |

- After having considered the above cases, there is nothing left to do but put the base root and modifier root together `{base}{modifier}

That's composition!
